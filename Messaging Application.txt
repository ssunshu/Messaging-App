Messaging Application Plans

Front-end: HTML5, CSS3, JavaScript, Angular, Node.js
Back-end: Express.js, SQL

Functionalities:
Models to be developed: Message & User
Users are able to create accounts
Users can upload messages up to 128 characters in length
Users will be able to upvote and downvote other user's messages but not their own
Users can delete their own messages but not other users
Each page will display up to 5 messages

Technologies:
HTML5 and CSS3 are utilized for web layouts and designs
Angular is used for working framework
JavaScript is used for scripting and backend coding
Express.js will be utilized for REST API calls
Database will utilize SQLite. All data will be stored remotely on server
Node.js is utilized for package installations
